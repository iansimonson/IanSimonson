About the site

This is my personal site to talk about technologies I'm experimenting with or other similar projects. However I've only recently decided to get this back up and running so for now it is mostly just contact info. Currently the site is just static pages using Nginx though I plan on experimenting with Rocket or something more custom using Zig. The site's layout may also change.

I've learned and continue to learn by googling, reading blog posts, and copious amounts of copying, splicing, and sanitizing various code snippets from other people. As such, all source code for this site and any code snippets I might post here will be on GitHub.

The site's source is available on GitHub currently under the WTFPL but after some research will be under the Zero-Clause BSD or CC-0 which hopefully is more useful
